# Banner Images

## Hướng dẫn sử dụng Banner

### Kích thước khuyến nghị:
- **Desktop**: 1200 x 400 px
- **Tablet**: 800 x 350 px  
- **Mobile**: 600 x 300 px

### Format file:
- JPG/JPEG (nén tốt, dung lượng nhỏ)
- PNG (chất lượng cao, hỗ trợ trong suốt)
- WebP (hiện đại, dung lượng nhỏ nhất)

### Cách thêm banner mới:

1. Chuẩn bị ảnh banner với kích thước phù hợp
2. Đặt tên file rõ ràng (VD: banner-product-name.jpg)
3. Copy file vào thư mục `/images/banners/`
4. Cập nhật đường dẫn trong file `index.html` hoặc `js/main.js`

### Lưu ý:
- Dung lượng file nên < 200KB để tải nhanh
- Sử dụng công cụ nén ảnh: TinyPNG, Squoosh
- Đảm bảo văn bản trên banner rõ ràng, dễ đọc
- Kiểm tra hiển thị trên cả desktop và mobile
